# 🔧 Auto GUI Scale

This mod changes the auto GUI value by reducing it by 1.

## ✨ Features

- Changes the auto GUI value by reducing it by 1.

## 🔄 Versions

- Minecraft  >= 1.20
- Fabric >= 0.16.4
- Quilt  >= 0.17.3

# ⬇️ Installation

1. Download the mod from [Modrinth](https://modrinth.com/project/Un1BZCMR) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/auto-gui-scale)
2. Place it into your `mods` directory
